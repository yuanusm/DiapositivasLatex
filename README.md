# Diapositivas LaTeX: Guía de Estadística Descriptiva

Este repositorio contiene una presentación en LaTeX Beamer para estudiar conceptos de Estadística Descriptiva que suelen confundirse entre sí.

## Archivo principal

- `guia_estadistica_descriptiva.tex`: presentación completa con diapositivas sobre universo, población, muestra, censo, variables, escalas de medición, frecuencias, intervalos de clase y tablas de contingencia.

## Compilación

Para generar el PDF, ejecuta:

```bash
pdflatex guia_estadistica_descriptiva.tex
```

Si tu distribución de LaTeX lo requiere, ejecuta el comando dos veces para estabilizar referencias y numeración.
